# 🚀 Setup Zig di Neovim (NvChad)

Panduan ini menjelaskan cara setup environment **Zig** di **Neovim** menggunakan **NvChad**.

## 1. Requirements

### Core

* **Neovim** (disarankan versi terbaru)
* **Zig** (stable terbaru)

### Tools Pendukung

* **Nerd Font** → *FiraCode Nerd Font*
* **Tree-Sitter CLI**
* **ripgrep** (search super cepat di Neovim)

## 2. Install Zig

### Via VUR(T4n Labs)

* [Zig : 0.15.2](https://t4n-labs.github.io/vur-web/#package/zig)

### Manual Install

```bash
wget https://ziglang.org/download/.../zig.tar.xz
tar -xf zig.tar.xz
sudo mv zig /opt/zig
```

Tambahkan ke PATH (`~/.bashrc` / `~/.zshrc`):

```bash
export PATH="/opt/zig:$PATH"
```

Cek instalasi:

```bash
zig version
```

## 3. Setup LSP (ZLS)

### Install ZLS (Zig Language Server)

Di dalam Neovim:

```vim
:MasonInstall zls
```

### Konfigurasi LSP Dasar

Edit file:

```
~/.config/nvim/lua/config/lspconfig.lua
```

Isi:

```lua
local servers = { "zls" }
```

### ⚡ Konfigurasi Lanjutan (Opsional tapi Disarankan)

Edit:

```
~/.config/nvim/lua/custom/configs/lspconfig.lua
```

```lua
local lspconfig = require("lspconfig")

lspconfig.zls.setup({
  settings = {
    zls = {
      enable_inlay_hints = true,
      enable_snippets = true,
      warn_style = true,
    },
  },
})
```

## 4. Setup Tree-Sitter

Install parser Zig:

```vim
:TSInstall zig
```

Cek status:

```vim
:TSModuleInfo
```

## 🎨 5. Format & Style

Zig sudah memiliki formatter bawaan:

```bash
zig fmt main.zig
```

### Auto Format di Neovim

Tambahkan ke:

```
~/.config/nvim/lua/custom/autocmds.lua
```

```lua
vim.api.nvim_create_autocmd("BufWritePre", {
  pattern = "*.zig",
  callback = function()
    vim.cmd("silent! !zig fmt %")
    vim.cmd("edit")
  end,
})
```

## 6. Debugging (Opsional)

Zig belum memiliki debugging se-smooth bahasa lain, tapi bisa menggunakan:

* `lldb`
* `gdb`

Install:

```bash
sudo xbps-install -S lldb
```

## 7. Troubleshooting

### ZLS tidak jalan

Cek:

```vim
:LspInfo
```

Jika tidak terdeteksi:

* Pastikan `zls` sudah terinstall
* Pastikan ada di PATH:

```bash
which zls
```

### Tree-sitter error

```vim
:TSUpdate
```

### NvChad bermasalah

```vim
:Lazy sync
:MasonUpdate
```

## 8. Reload Config

Setelah konfigurasi:

```vim
:Lazy sync
```

atau restart Neovim:

```bash
nvim
```
