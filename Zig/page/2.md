# Hello World

Hello World merupkan program yang populer untuk sesorang yang barumemulai belajar.

##### Langkah Membuat Hello World Zig
1. Buat File dengan ekstensi `.zig`. contoh : hello.zig
2. Tulis Kode Hello World :
    ```zig
    const std = @import("std");

    pub fn main() !void {
        const stdout = std.io.getStdOut().writer();
        try stdout.print("Halo, dunia!\n", .{});
    }
    ```
    - `print()` → Fungsi untuk menampilkan teks ke layar
    - `"..."` → String literal (teks)
    - `try` → Menangani error (error propagation)
3. Simpan File.
4. Jalankan Di Terminal :
    ```bash
    zig run hello.zig
    ```
